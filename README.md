# Computer Vision Projects

This repository contains machine learning projects built using **TensorFlow** and **Keras**. Each project utilizes Convolutional Neural Networks (CNNs) for binary image classification.

---

## Project 1: Pen vs. Pencil
A model trained to distinguish between images of pens and pencils.

### Model Performance
The graph below shows the training accuracy over 25 epochs:

![Pen vs Pencil Accuracy](PENCIL%20VS%20PEN/accuracy_graph.png)

### File Structure
* **Code:** `PENCIL VS PEN/program.ipynb`
* **Model:** `PENCIL VS PEN/final_model.h5`
* **Dataset:** `PENCIL VS PEN/split_dataset/`
